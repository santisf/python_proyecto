Proyecto Django: Adopcion

Este proyecto es una aplicación web desarrollada con Django para gestionar adopciones de animales. Los usuarios pueden explorar animales disponibles, ver detalles, enviar solicitudes de adopción y gestionar su perfil.

Funcionalidades principales
Página de inicio
Vista principal con buscador por nombre o especie.

Gestión de Animales
Listado de animales publicados.
Visualización del detalle de cada animal.
Creación, edición y eliminación de animales (solo usuarios autorizados).
Gestión de Solicitudes
Creación y listado de solicitudes de adopción.
Gestión de Usuarios
Registro de nuevos usuarios.
Login y Logout.
Perfil de usuario con posibilidad de edición y cambio de contraseña.
Sistema de mensajes para notificaciones entre usuarios y administrador.

Cómo ejecutar el proyecto
Clonar el repositorio.

Crear y activar un entorno virtual.

Instalar las dependencias:

Código
pip install -r requirements.txt
Ejecutar las migraciones:

Código
python manage.py migrate
Iniciar el servidor de desarrollo:

Código
python manage.py runserver
